Завдання №2

Умова:

Змінна day може набувати одне із 7 значень: 1, 2, 3, 4, 5, 6, 7. Це значення потрібно запитати у користувача через prompt.

Якщо змінна має значення 1, то у змінну finish потрібно записати понеділок, якщо має значення 2 — вівторок і так далі.

  • Розв’язати завдання через switch-case.
  • Додати default на випадок, якщо користувач введе в prompt щось інше.
  • Значення змінної finish вивести в консоль.

Код JS:


let day = Number(prompt("Введіть номер дня тижня"));
    console.log(`Введений номер дня тижня: ${day}`);
    let finish;

    switch (day) {
        case 1:
            finish = "Понеділок";
            break;
        case 2:
            finish = "Вівторок";
            break;
        case 3:
            finish = "Середа";
            break;
        case 4:
            finish = "Четвер";
            break;    
        case 5:
            finish = "П'ятниця";
            break; 
        case 6:
            finish = "Субота";
            break;
        case 7:
            finish = "Нелідя";
            break;
        default:
            finish = "Неправильний номер дня тижня";
    }
console.log(`День тижня: ${finish}`);
alert(`День тижня: ${finish}`);
