Умова:

Оголосити функцію makeArray(firstArray, secondArray, maxLength), яка створює та повертає новий масив.

Функція приймає три параметри:

  • firstArray — перший масив;
  • secondArray — другий масив;
  • maxLength — максимальна довжина нового масиву.

Функція повинна створити новий масив, який містить усі елементи з firstArray, а потім усі елементи з secondArray.

Якщо кількість елементів у новому масиві перевищує maxLength, функція повинна повернути копію масиву довжиною maxLength елементів.

В іншому випадку функція повинна повернути весь новий масив.

Код JS:


function makeArray(firstArray, secondArray, maxLength ) {

    let newArray = firstArray.concat(secondArray);

    if(newArray.length >  maxLength){
        return newArray.slice(0, maxLength);
    }

    else{
        return newArray;
    }
}
console.log(makeArray([1, 2], [3, 4], 3));
alert(makeArray([1, 2], [3, 4], 3));

console.log(makeArray([5, 6, 7], [8, 9], 5));
alert(makeArray([5, 6, 7], [8, 9], 5));
