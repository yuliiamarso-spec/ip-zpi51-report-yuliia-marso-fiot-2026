Оголосити функцію labTask9ObjArray(), яка виконує обробку об’єкта concerts з інформацією про дати концертів у різних містах.

У функції необхідно:

  • створити об’єкт concerts, який містить назви міст та дати концертів;
  • отримати з об’єкта масив, який буде містити лише назви міст;
  • прибрати з масиву міста, в яких концерт уже пройшов;
  • відсортувати міста у хронологічному порядку відповідно до дат концертів;
  • вивести результат у консоль.

Для виконання завдання необхідно застосувати стрілочні функції.

Очікуваний результат: ["Одеса", "Умань", "Харків"]

const concerts = {

};

        }
    }
}
