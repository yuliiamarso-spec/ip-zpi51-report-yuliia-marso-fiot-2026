Завдання №3

Умова:

Напишіть функцію, яка приймає масив об’єктів medicines та повертає новий масив об’єктів, у якому для всіх медикаментів із ціною більше 300 грн застосовується знижка 30%.

У новому масиві необхідно:

  • додати для кожного медикамента id;
  • залишити назву медикамента;
  • змінити ціну з урахуванням знижки 30%.

Результат роботи програми необхідно вивести у консоль.

Для виконання завдання необхідно використати методи filter() та map(), а також стрілочні функції.

Код JS:


function getDiscounts () {
    const medicines = [
        {name: "Noshpa", price: 170 },
        {name: "Analgin", price: 55 },
        {name: "Quanil", price: 310 },
        {name: "Alphacholine", price: 390 },
    ];


const filterPrices = medicines.filter(medicine => medicine.price > 300);

const medicinesDiscounts = filterPrices.map((medicine, index) => ({
    id: index + 1,
    name: medicine.name,
    price: medicine.price * 0.7
}))

console.log(medicinesDiscounts);

}
getDiscounts();
