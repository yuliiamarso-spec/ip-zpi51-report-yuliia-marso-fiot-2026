Умова:

Оголосити функцію getProductDetails(productId, successCallback, errorCallback), яка отримує інформацію про товар за його ідентифікатором.

Функція приймає три параметри:

  • productId — ідентифікатор товару;
  • successCallback — колбек-функція, яка викликається у випадку успішного пошуку товару;
  • errorCallback — колбек-функція, яка викликається у випадку помилки.

У функції необхідно:

  • виконати пошук товару за його ідентифікатором;
  • якщо товар знайдено — передати інформацію про нього у successCallback;
  • якщо товар не знайдено — викликати errorCallback та передати повідомлення про помилку.

У випадку успішного пошуку функція повинна передати об’єкт товару, який містить його ідентифікатор, назву та ціну.

Якщо товар не знайдено, функція повинна передати повідомлення:

Помилка: товар не знайдено.

Код JS:


function getProductDetails (productId, successCallback, errorCallback) {
    const products = [
        { id: 1, name: "Ноутбук", price: 25000 },
        { id: 2, name: "Смартфон", price: 15000 },
        { id: 3, name: "Навушники", price: 3000 }
    ];

    const product = products.find(item => item.id === productId);

    if (product) {
        successCallback(product);
    } else {
        errorCallback("Помилка: товар не знайдено.");
    }
}

    function showProduct(product) {
        console.log("Товар знайдено:");
        console.log(product);
    }


    function showError(message) {
        console.log(message);
    }

getProductDetails(2, showProduct, showError);
getProductDetails(10, showProduct, showError);
