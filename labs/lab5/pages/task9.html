Умова:

Дано масив об’єктів users, який містить інформацію про користувачів.

Кожен об’єкт містить:

  • name — ім’я користувача;
  • age — вік користувача.

Необхідно відсортувати масив об’єктів за віком користувачів у порядку зростання.

Очікуваний результат:

[{name: "Bob", age: 19}, {name: "John", age: 27}, {name: "Jane", age: 31}]

Результат роботи програми необхідно вивести у консоль.

Код JS:


function sortUsersByAge(){ 

    const users = [
        { name: "John", age: 27 },
        { name: "Jane", age: 31 },
        { name: "Bob", age: 19 },
    ];

    const result = users.sort((a, b) => a.age - b.age)

    return result;
}

console.log(sortUsersByAge());
