Завдання №8

Умова:

Дано масив чисел numbers.

Необхідно створити новий масив, який міститиме квадрати кожного елемента вхідного масиву.

Очікуваний результат:

[1, 4, 9, 16, 25]

Результат роботи програми необхідно вивести у консоль.

Для виконання завдання необхідно використати метод map() та стрілочні функції.

Код JS:


function squareNumbers(){ 

const numbers = [1, 2, 3, 4, 5];

const result = numbers.map(number => number * number);

    return result;
}

console.log(squareNumbers());
