Умова:

У звітному HTML-документі створити форму логіна form.login-form.

Форма повинна містити два поля введення: email та password, а також кнопку відправлення форми.

Необхідно написати скрипт керування формою логіна. Обробка відправлення форми повинна відбуватися за подією submit.

Під час відправлення форми сторінка не повинна перезавантажуватися.

Якщо при відправленні форми є незаповнені поля, необхідно вивести alert з повідомленням: "All form fields must be filled in".

Валідація повинна відбуватися саме через JavaScript, без додавання атрибута required до інпутів.

Якщо користувач заповнив усі поля і відправив форму, необхідно зібрати значення полів в об’єкт з двома властивостями. Ключами об’єкта мають бути імена інпутів, а значеннями — відповідні значення, очищені від пробілів по краях.

Для доступу до елементів форми необхідно використати властивість elements.

Після відправлення форми об’єкт із введеними даними потрібно вивести в консоль, а значення полів форми очистити методом reset().

Код HTML:


<div class="task-box">

    <h3>ЗАВДАННЯ 5</h3>

    <form class="login-form">

        <label>
            Email
            <input type="email" name="email" />
        </label>

        <label>
            Password
            <input type="password" name="password" />
        </label>

        <button type="submit">Log in</button>

    </form>

</div>

Код JS:


const form = document.querySelector(".login-form");

form.addEventListener("submit", function (event) {
    event.preventDefault();

    const email = form.elements.email.value.trim();
    const password = form.elements.password.value.trim();

    if (email === "" || password === "") {
        alert("All form fields must be filled in");
        return;
    }

    const formData = {
        email: email,
        password: password
    };

    console.log(formData);
    form.reset();
});
