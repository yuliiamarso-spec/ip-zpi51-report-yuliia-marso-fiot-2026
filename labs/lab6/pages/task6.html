Завдання №6

Умова:

Написати скрипт, який змінює колір фону елемента body через інлайн-стиль після натискання на кнопку.

У HTML-документі створити блок div.widget, який містить текстовий елемент для відображення поточного кольору та кнопку зміни кольору.

Необхідно:

  • знайти кнопку з класом change-color;
  • знайти елемент span.color;
  • додати обробник події click на кнопку;
  • після натискання згенерувати випадковий колір;
  • змінити колір фону body на згенерований колір;
  • записати значення кольору текстом у span.color.

Для генерації випадкового кольору необхідно використати функцію getRandomHexColor().

Результат виконання:

ЗАВДАННЯ 6

Background color: -

Код HTML:


<div class="task-box">

    <h3>ЗАВДАННЯ 6</h3>

    <div class="widget">
        <p>Background color: <span class="color">-</span></p>
        <button type="button" class="change-color">Change color</button>
    </div>

</div>

Код JS:


const button = document.querySelector("button.change-color");    
const colorText = document.querySelector("span.color");
const taskBox = document.querySelector(".task-box");


button.addEventListener("click", () => {

    const color = getRandomHexColor();
    taskBox.style.backgroundColor = color;        
    colorText.textContent = color;

});

function getRandomHexColor() {
    return `#${Math.floor(Math.random() * 16777215)
        .toString(16)
        .padStart(6, 0)}`;
}
