Завдання №2

Умова:

У звітному HTML-документі створити HTML-розмітку, яка складається з тексту, блоку (div) та двох кнопок.

Кнопка Зменшити повинна зменшувати розмір квадрата на 15 пікселів.

Кнопка Збільшити повинна збільшувати розмір квадрата на 15 пікселів.

Повторне натискання на кнопки повинно змінювати розмір квадрата відповідно до обраної дії.

Результат виконання:

ЗАВДАННЯ 2

Код HTML:




<div class="task-box">

    <h3>ЗАВДАННЯ 2</h3>

    <div class="task2-square"></div>

    <div class="task-controls">

        <button id="decreaseBtn">Зменшити</button>

        <button id="increaseBtn">Збільшити</button>

    </div>

</div>

Код JS:


const square = document.querySelector(".task2-square");
const decreaseBtn = document.querySelector("#decreaseBtn");
const increaseBtn = document.querySelector("#increaseBtn");

let size = 100;

decreaseBtn.addEventListener("click", function () {
    size = size - 15;
    square.style.width = size + "px";
    square.style.height = size + "px";

});

increaseBtn.addEventListener("click", function () {
    size = size + 15;
    square.style.width = size + "px";
    square.style.height = size + "px";

});
