Завдання №1

Умова:

У звітному HTML-документі створити HTML-розмітку, яка складається з тексту, кнопки та двох інпутів.

Після натискання на кнопку SWAP ME повинен здійснюватися обмін вмістом між двома інпутами.

Результат виконання:

ЗАВДАННЯ 1

Код HTML:


<div class="task-box task1-box">

    <h3>ЗАВДАННЯ 1</h3>

    <div class="task1-controls">

        <button id="swapBtn">SWAP ME</button>

        <input id="firstInput" type="text" value="???">

        <input id="secondInput" type="text" value="!!!">

    </div>

</div>

Код JS:


const swapBtn = document.querySelector("#swapBtn");
const firstInput = document.querySelector("#firstInput");
const secondInput = document.querySelector("#secondInput");

swapBtn.addEventListener("click", function () {
    const temp = firstInput.value;

    firstInput.value = secondInput.value;
    secondInput.value = temp;
});
