Завдання №3

Умова:

У звітному HTML-документі створити HTML-розмітку, яка складається зі списку елементів та кнопки.

Після натискання на кнопку Подвоїти значення кожного елемента списку повинне збільшуватися у два рази.

Результат виконання:

ЗАВДАННЯ 3

  • 1
  • 4
  • 8
  • 16
  • 20
  • 30

Код HTML:




<div class="task-box">

    <h3>ЗАВДАННЯ 3</h3>

    <div class="task3-list">
        <ul>
            <li>1</li>
            <li>4</li>
            <li>8</li>
            <li>16</li>
            <li>20</li>
            <li>30</li>
        </ul>
    </div>

    <div class="task-controls">

        <button id="doubleBtn">Подвоїти</button>

    </div>

</div>

Код JS:


const list = document.querySelector(".task3-list");
const items = document.querySelectorAll(".task3-list li");
const double = document.querySelector("#doubleBtn");

doubleBtn.addEventListener("click", function () {

    items.forEach(item => {
        item.textContent = item.textContent * 2;
    });

});
